README
======

This package includes the following files.

|-- Driver.java [File containing the main() method]
|-- Buffer.java [The buffer class that utilizes wait() and notifyAll() to synchronize Producer and Consumer]
|-- Producer.java [The producer, implements Runnable]
|-- Consumer.java [The consumer, implements Runnable]
|-- Makefile [A Makefile which carries out the operations all, run, and clean]
|-- README.txt [This file]

To compile:
    make all

To run:
    make run
